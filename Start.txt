EcoEYE local start guide

Backend
1. Open a terminal in the project root.
2. Start the Flask app:
	.\.venv\Scripts\python.exe app.py
3. The backend runs on http://127.0.0.1:80

Frontend
1. Open a second terminal.
2. Start the Vite app:
	cd control-app
	npm run dev -- --host 127.0.0.1
3. The frontend runs on http://127.0.0.1:5173

Notes
- The frontend proxies /api requests to the backend at http://127.0.0.1:80.
- If dependencies are missing, install them before starting the apps.
